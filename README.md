# 🧬 DNA Nucleotide Counter

A Python program that analyzes DNA sequences, counts nucleotides, and detects invalid characters entered by the user.

## Features

- Counts the number of A, T, G, and C nucleotides
- Accepts DNA sequences entered by the user
- Converts lowercase letters to uppercase automatically
- Detects and reports invalid characters
- Built using Python dictionaries, loops, and conditionals

## How to Use

Run the program:

```python
python dna_nucleotide_counter.py
```

Enter a DNA sequence when prompted:

```text
Enter a DNA sequence:
ATTAXPCGGC
```

Example output:

```text
Warning: Unknown nucleotide detected: X
Warning: Unknown nucleotide detected: P

Nucleotide counts:
A: 2
T: 2
G: 2
C: 2
```
